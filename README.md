# Mess Around — The Ultimate Distraction Hider

This is a modernized remake of the archived repository [MessAround](https://github.com/rhzxg/MessAround), now integrated into the [Revelation](https://github.com/revelationtodo/Revelation) TODO application as a plugin.

## How to Use

1. Download the Revelation TODO application from the [Releases](https://github.com/revelationtodo/Revelation/releases) page.
2. Download the Mess Around plugin from the release page of this repo.
3. Extract the downloaded `.zip` file into the Revelation installation directory.
4. Launch `Revelation.exe` and navigate to the **Mess Around** plugin page.
5. Drag the “+” button to pick a window you want to hide or show.
6. Set a custom hotkey(Pause by default) to toggle that window’s visibility.
7. Press the hotkey to hide or show the selected window at any time.



# Mess Around — 摸鱼利器

这是对已归档仓库 [MessAround](https://github.com/rhzxg/MessAround) 的全新升级版，并以插件形式集成到 [Revelation](https://github.com/revelationtodo/Revelation) TODO软件中。

## 使用说明

1. 从 [Releases](https://github.com/revelationtodo/Revelation/releases) 页面下载 Revelation TODO 软件。
2. 从插件的发布页面下载 Mess Around 插件。
3. 将下载的 `.zip` 解压到 Revelation 的安装目录下。
4. 启动 `Revelation.exe`，切换到 **Mess Around** 插件页面。
5. 拖拽 “+” 按钮，选择你希望隐藏/显示的窗口。
6. 设置一个热键（默认为Pause），用于切换该窗口的可见性。
7. 按下或松开热键即可随时隐藏或显示该窗口。
